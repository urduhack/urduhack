Installation
============

Notes
-----
.. note:: Urduhack is supported on the following Python versions

+--------------+-------+-------+-------+-------+-------+
|**Python**    |**2.7**|**3.4**|**3.5**|**3.6**|**3.7**|
+--------------+-------+-------+-------+-------+-------+
|Urduhack      |       |       |       |  Yes  |  Yes  |
+--------------+-------+-------+-------+-------+-------+

Basic Installation
------------------
Install Urduhack with :command:`pip`::

    $ pip install Urduhack
