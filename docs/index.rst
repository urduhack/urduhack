Welcome to Urduhack Documentation!
==================================

What is Urduhack?
-----------------

Urduhack is an **open-source** NLP library for urdu language. It comes with a lot of battery included features to help
you process Urdu data in the easiest way possible.

Our Goal
--------

- **Academic users** Easier experimentation to prove their hypothesis without coding from scratch.
- **NLP beginners** Learn how to build an NLP project with production level code quality.
- **NLP developers** Build a production level application within minutes.

**Urduhack** is maintained by `Ikram Ali and Contributors <https://github.com/akkefa>`_.
